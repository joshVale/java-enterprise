# Java Basics and Array Manipulation – Lesson Project

## 📌 Описание
Проект демонстрирует основные навыки работы с Java:
- Методы и их параметры  
- Условные конструкции (`if`, `else`)  
- Циклы (`for`)  
- Работа с одномерными и двумерными массивами  
- Логические проверки (`boolean`)  
- Статические методы и организация кода в классах  

Проект состоит из двух классов:
1. `Main` – демонстрация работы методов.
2. `Lesson2` – набор методов для работы с массивами и числами.

Этот проект может использоваться как учебный пример для начинающих Java-разработчиков или для подготовки к собеседованиям на junior позиции.

## 📚 Методы проекта

### Main
- `printThreeWords()` – выводит три слова: `Orange, Banana, Apple`.
- `checkSumSign()` – проверяет сумму двух чисел и выводит положительная или отрицательная.
- `printColor()` – проверяет значение переменной и выводит `Красный`, `Желтый` или `Зеленый`.
- `compareNumbers()` – сравнивает два числа и выводит результат.
- `check(int a, int b)` – проверяет, находится ли сумма `a+b` в диапазоне [10, 20).
- `positiveOrNegative(int t)` – выводит, положительное или отрицательное число.
- `checkPositiveOrNegative(int t)` – возвращает `true`, если число положительное, иначе `false`.
- `printString(String s, int k)` – выводит строку `s` `k` раз.
- `isYear(int y)` – проверяет, является ли год високосным.

### Lesson2
- `changeZerotoOne(int[] arr)` – меняет все 0 на 1 и 1 на 0 в массиве.
- `printArr()` – создаёт массив длиной 100 и выводит числа от 1 до 100.
- `giveArr(int[] num)` – удваивает элементы массива, которые меньше 6, и возвращает изменённый массив.
- `fillDiagonals(int[][] arrays)` – заполняет главную и побочную диагонали двумерного массива единицами и выводит его.
- `ArrValue(int len, int initialValue)` – создаёт массив длины `len`, заполняет его значением `initialValue` и возвращает массив.
- `minMax(int[] num)` – находит и выводит минимальный и максимальный элемент массива.
- `checkBalance(int[] arr)` – проверяет, существует ли индекс, где сумма левой и правой части массива равна; возвращает `true` или `false`.
